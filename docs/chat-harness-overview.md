# FL Chat Harness — Requirements (Ext JS Aligned)

## Overview
Chat will be implemented using Ext JS within the existing FusionLive UI.

## Phase 1
- Workspace Chat only
- Admin-based indexing — **Administration** nav-tab → **Company Information** → **AI Indexing** tile.
  Admin view follows the standard FusionLive admin layout: blue *Administration* tab + *AI Indexing Configuration* sub-heading above the grid. Priority Controls card has four icon-only buttons: move to top, up, down, move to bottom. See [workspace-index-governance.md](workspace-index-governance.md).
