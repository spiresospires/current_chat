# Commands & Interaction

## 🎯 Job Story
When using Chat frequently,
I want quick command access,
so that I can interact efficiently.

---

# ✅ Functional Requirements

## FR1 — Slash Commands
The system shall support:
- slash command interaction

---

## FR2 — Menu Parity
The ⋮ menu shall:
- expose equivalent functionality

---

## FR3 — Command Behaviour
Commands shall:
- trigger actions
- not automatically submit queries

---

# ✅ Acceptance Criteria

## AC1 — Slash Command
Given the user enters a slash command
Then the matching action is triggered

---

## AC2 — Menu
Given the user opens the menu
Then equivalent actions are available
